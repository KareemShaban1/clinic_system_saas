1- admin guard
2- clinic guard
3- patient guard

- admin can add / edit / delete / show governorates
- admin can add / edit / delete / show cities

- admin can add / edit / delete / show subscriptions
- admin can add / edit / delete / show clinics
- admin can add / edit / delete / show events
